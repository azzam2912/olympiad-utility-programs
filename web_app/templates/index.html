Welcome to Olympiad Utility Programs

This web application provides a collection of tools to help with various Olympiad-related tasks. Select a tool from the categories below to get started.

PDF Tools

Tools for manipulating and extracting content from PDF files.

  • Extract Images
  • Extract Questions
  • Split PDF
  • Remove Headers/Footers
Open PDF Tools

Olympiad File Renamer

Rename Olympiad PDF files to a standardized format for better organization.

This tool helps you batch rename Olympiad-related files according to a consistent naming convention, making it easier to organize and find files later.

Open File Renamer

Picture Renamer

Rename pictures based on a names.txt file, useful for organizing participant photos.

Upload a zip file containing pictures and a names.txt file with the desired names, and this tool will rename all the pictures accordingly.

Open Picture Renamer

Web PDF Scraper

Download PDFs from web sources like competition websites.

Supports universal scraping and specialized scraping for KOMA competition PDFs. Simply provide the URL and choose the scraper type.

Open Web Scraper
