How It Works

1. Image Analysis

The tool analyzes each image to identify competition information.

2. Pattern Recognition

It detects patterns that indicate the competition type, year, and problem number.

3. Automatic Renaming

Images are renamed according to a standardized format: [Competition]_[Year]_[Problem].jpg
