This tool helps you download PDFs from web sources like competition websites.

Enter the URL of the webpage containing PDF links, and choose the appropriate scraper type. The tool will download all PDFs found on the page.

How It Works

1. URL Analysis

The tool visits the provided URL and analyzes the page content.

2. PDF Detection

It identifies all PDF links on the page based on the selected scraper type.

3. Automatic Download

The tool downloads all detected PDFs to the configured download directory.

Scraper Types

  • Universal Scraper: Works with most websites by detecting links ending with .pdf
  • KOMA Scraper: Specialized for the KOMA competition website structure, which may have PDFs in specific locations or formats
