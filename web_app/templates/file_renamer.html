How It Works

1. File Detection

The tool scans the configured directory for PDF files that match Olympiad naming patterns.

2. Pattern Recognition

It analyzes each filename to extract information like year, country, competition level, and language.

3. Standardized Renaming

Files are renamed to follow a consistent pattern, making them easier to organize and find.

4. Verification

The tool verifies each rename operation and reports success or failure.
